# Advanced Fluid Infrastructure

Advanced Fluid Infrastructure is a Factorio 2.0 + Space Age mod that owns the fluid infrastructure progression split out from Advanced Electric Remade.

It provides upgraded pipes, pipe-to-ground, pumps, offshore pumps, pipeline extent progression, fluid infrastructure technologies, environment-specific fluid branches, production-machine fluidbox extent patches, and fluid compatibility cleanup.

Advanced Electric Remade should depend on this mod for fluid infrastructure prototypes while retaining ownership of power generation entities and their power-specific fluidbox tuning.

Current tier behavior is documented in [docs/fluid-infrastructure-benchmark.md](docs/fluid-infrastructure-benchmark.md).

Release packaging and automated deployment are documented in [docs/release-process.md](docs/release-process.md).
